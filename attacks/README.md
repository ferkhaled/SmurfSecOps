# Attack Simulation Notes

These exercises are for safe learning inside your own lab only.
They are meant to help `papa-sec` observe what `gargamel` might try.

## Safety principles

- use only lab systems you control
- record the baseline state before each exercise
- make one change at a time
- reset the environment after the test

## Starter scenarios

- `container-shell.md`
- `suspicious-network-activity.md`
- `vulnerable-image.md`

## Suggested workflow

1. confirm the app is healthy
2. run one small scenario
3. observe logs, events, or Falco alerts
4. discuss how to harden or detect the behavior better
