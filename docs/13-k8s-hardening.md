# 13 - Kubernetes Hardening

Kubernetes security gets easier when you build habits early.
This repo includes a few small examples so you can discuss hardening without overwhelming the first version.

## Starter hardening topics

- least-privilege RBAC
- Pod Security admission labels
- NetworkPolicy basics
- non-root containers
- read-only filesystems as a later step

## Files to explore

- `security/kubernetes/hardening-checklist.md`
- `security/kubernetes/network-policy-example.yaml`
- `security/kubernetes/pod-security-example.yaml`
- `security/kubernetes/rbac-example.yaml`

These examples are intentionally small and should be adapted carefully before real use.
