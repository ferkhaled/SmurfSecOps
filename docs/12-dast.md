# 12 - DAST

Dynamic Application Security Testing sends requests to a running application and looks for unsafe behavior.

## Tool used here

- OWASP ZAP baseline scan

## Starter scope

The first version focuses on safe baseline checks against a running BerryShop API endpoint.

## Why this matters

- it helps learners see the difference between code scanning and runtime testing
- it encourages basic security validation before release

See `security/zap/README.md` for starter usage.
