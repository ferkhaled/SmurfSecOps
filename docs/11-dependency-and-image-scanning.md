# 11 - Dependency And Image Scanning

Dependency and image scanning help you catch known vulnerable packages and container layers.

## Tool used here

- Trivy

## Starter use cases

- scan the Python dependency tree
- scan the built BerryShop image
- scan Kubernetes manifests for risky settings

## Learning goal

Treat scan results as a conversation starter.
Not every finding needs the same response, but every finding deserves review.

See `security/trivy/README.md` for sample commands.
